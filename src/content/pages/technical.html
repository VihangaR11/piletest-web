09 / Frequently asked questions

Technical Resource FAQ

Quick guidance for document requests and early test planning.

Ask a technical question →
Can I download method statements directly?

Method statements are supplied on request so the current revision and project-specific preparation requirements can be confirmed.

Which testing standard will apply to my project?

The project specification, contract requirements and current applicable standard should be confirmed with the consultant and testing team before work begins.

What information should I include with a request?

Provide the service required, foundation type, pile dimensions, anticipated test load, site location, programme and the relevant specification where available.

Are published system capacities guaranteed for every site?

No. Equipment range is only one factor. The achievable test arrangement depends on reaction design, access, pile geometry, ground conditions and project controls.

Can the technical team review a proposed testing scope?

Yes. Submit the available drawings, specification and performance requirements for a project-specific technical review.
